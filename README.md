# mmood

This project discovers latent music mood groups using unsupervised clustering.
Songs or listening sessions are grouped based on numeric audio and behavior features,
without any predefined labels.

## Keywords
- Unsupervised Learning
- Music Mood Analysis
- Clustering

## Dataset Format
Input CSV files must contain the following numeric columns:

feature1, feature2, feature3, feature4, feature5

Two datasets (dataset_A.csv and dataset_B.csv) are merged automatically.
